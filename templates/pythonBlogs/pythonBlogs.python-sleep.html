PYTHON SLEEP FUNCTION

sleep() is defined in time() module of python. It Provides useful functions to handle time-related tasks. The sleep() suspends execution for the given number of seconds.For example you are waiting for a process to upload the file completely. 

import time
print("Hi sfe")
time.sleep(5)
print("After 5 seconds sfe prints")
Hi sfe
After 5 seconds sfe prints

Working of sleep() function ...
To understand the working of sleep() function, we'll print the current execution time before using the sleep() function and execution time after using the sleep() function. 

import time
print("Before sleep(): execution time= ",time.ctime())
time.sleep(5)
print("After sleep(): execution time= ",time.ctime())
Before sleep(): execution time = Mon Jun 14 14:29:28 2021
After sleep() : execution time = Mon Jun 14 14:29:33 2021
